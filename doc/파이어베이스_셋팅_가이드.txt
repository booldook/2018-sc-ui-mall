Firebase 설치
1. nodejs.org 에서 node.js를 다운로드후 설치한다.
2. cmd 창을 열고 node -v 로 설치를 확인한다.
3. npm -g install firebase-tools 를 입력후 엔터.

Firebase 셋팅순서
1. 구글 Firebase에서 프로젝트를 생성한다.
2. 자신의 작업공간에 프로젝트 루트를 만든다(예:site2)
3. vsc에서 터미널 창을 열고 본인의 프로젝트 루트에서 firebase init 을 실행시킨다.
4. 설치 완료후 firebase serve 를 실행하여 본인의 컴퓨터에서 작업할 준비를 마친다.
